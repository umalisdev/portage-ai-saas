Environnement Technique Optimisé

Image Docker unique multi-services basée sur Python 3.12 et Playwright

Dockerfile
# Base image légère et sécurisée FROM python:3.12-slim # Installation des dépendances système RUN apt-get update && apt-get install -y \ wget curl unzip xvfb libnss3 ... # Configuration de l'environnement WORKDIR /app ENV PYTHONUNBUFFERED=1 ENV DISPLAY=:99 # Installation des dépendances Python COPY requirements.txt . RUN pip install --no-cache-dir -r requirements.txt # Installation des navigateurs Playwright RUN playwright install chromium RUN playwright install-deps chromium # Copie du code source COPY . . # Script d'entrée multi-services ENTRYPOINT ["./entrypoint.sh"]

Python 3.12 + Django Backend

Dernière version stable de Python pour des performances optimales. Django gère l'ORM, l'admin et l'API interne.

Playwright + Chromium Scraping

Moteur d'automatisation moderne, plus rapide et fiable que Selenium. Gère le rendu JS complexe de LinkedIn.

Xvfb (Virtual Framebuffer) Display

Permet d'exécuter un navigateur graphique en mode headless sur un serveur sans écran (nécessaire pour éviter les détections).

Entrypoint Script Orchestration

Un seul script bash gère le démarrage conditionnel des services (init, daemon, crm, exporter) selon la commande passée.
