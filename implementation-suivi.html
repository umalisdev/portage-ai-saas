En attente
Déploiement de Twenty CRM (Docker), configuration des objets custom (Consultant, Mission, Entreprise), import des données existantes, configuration SMTP et webhooks.
schedule Semaines 1-2 — Prérequis : serveur VPS (4 vCPU, 8 Go RAM)

Phase 2 — Scraping & Enrichissement
En attente
Installation d'OpenOutreach, configuration des campagnes LinkedIn (profils cibles IT/Conseil), déploiement Fire-Enrich avec les 4 agents IA, connexion API vers Twenty CRM.
schedule Semaines 3-4 — Prérequis : Phase 1 terminée + compte LinkedIn dédié
OpenOutreach Fire-Enrich Firecrawl API

Phase 3 — Orchestration & Automatisation
En attente

n8n Webhooks API REST

ML Tuning Monitoring Formation

    Simulation complète : profil LinkedIn → CRM → email → suivi — validation des données à chaque étape
