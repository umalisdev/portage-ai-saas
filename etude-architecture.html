Brique 1 — OpenOutreach

Scraping LinkedIn
person_search

OpenOutreach

Outil self-hosted d'automatisation LinkedIn B2B open source
star 100+ stars GitHub
code Python, Playwright, Django
cloud_off Self-hosted
gavel Open Source

OpenOutreach est un outil d'automatisation LinkedIn B2B conçu pour fonctionner en mode daemon 24/7. Il utilise l'API Voyager de LinkedIn (non officielle) via Playwright pour scraper les profils, les qualifier avec un modèle de Machine Learning (Gaussian Process Classifier + BALD active learning), et envoyer des demandes de connexion personnalisées avec des templates Jinja2.

psychology Qualification ML
  • auto_graph
    Gaussian Process Classifier (GPC) — Modèle probabiliste qui apprend à distinguer les profils pertinents des non-pertinents à partir d'exemples labellisés
  • school
    BALD Active Learning — Bayesian Active Learning by Disagreement : sélectionne les profils les plus informatifs pour améliorer le modèle avec un minimum de labels
  • tune
    Apprentissage continu — Le modèle s'améliore automatiquement à chaque interaction (acceptation/refus de connexion)

Brique 2 — Fire-Enrich

Enrichissement IA
auto_fix_high

Fire-Enrich

Alternative open source à Clay — Enrichissement IA multi-agents
star 1 100+ stars GitHub
code Python, Firecrawl, LLM
cloud_off Self-hosted / Railway
gavel Open Source

Fire-Enrich transforme une simple liste d'emails en un dataset riche grâce à 4 agents IA spécialisés. Chaque agent utilise Firecrawl pour le web scraping et un LLM pour l'analyse et la structuration des données. Le coût par enrichissement est de 0,01 à 0,05$ en self-hosted (API OpenAI), contre 149$/mois pour Clay.

business Company Research Agent

Identifie l'industrie, la localisation, la taille de l'entreprise, le chiffre d'affaires estimé, et les informations clés du site web corporate.

account_balance Fundraising Intelligence Agent

Détecte les levées de fonds récentes, les investisseurs, les montants, et le stade de financement (Seed, Series A, B, etc.).

groups People & Leadership Agent

Identifie les fondateurs, les dirigeants clés, leurs parcours, et les décideurs pertinents pour la prise de contact.

memory Product & Technology Agent

Analyse la stack technique, les produits principaux, les technologies utilisées, et les besoins potentiels en consulting IT.

Twenty CRM

Le #1 CRM Open Source — Alternative moderne à Salesforce (YC S23)
star 40 000+ stars GitHub
code TypeScript, React, GraphQL
cloud Self-hosted ou Cloud
gavel AGPL-3.0

Twenty est le CRM open source le plus populaire au monde, soutenu par Y Combinator (batch S23) et fondé par une équipe française. Il offre une alternative crédible à Salesforce avec un modèle de données entièrement personnalisable, des API GraphQL et REST puissantes, et un écosystème d'extensions en pleine croissance.

view_column Vues personnalisables

Kanban, Table, Timeline — chaque utilisateur peut créer ses propres vues avec filtres et tri avancés.

api API GraphQL & REST

Intégration programmatique complète. Requêtes et mutations pour toutes les entités du CRM.

extension Extension Chrome

Capture de leads directement depuis LinkedIn ou tout site web vers le CRM en un clic.

data_object Objets & champs custom

Modèle de données flexible : créez vos propres entités et relations sans code.

smart_toy MCP Server

Intégration native avec les assistants IA via le protocole Model Context Protocol.
